For the next task, week 3 I learned how to use CSS in more depth such as using a flexbox and adding a second stylesheet to my work. The week 3 task had a lot more images and text and allowed for a good understanding on how flexbox's worked which was very useful. As well as this I learned how to use class selectors which allows me to apply a different style to certain tags, a class selector can either be used to keep consistency with designs or to create specific designs

For the fourth week we went further into flexbox as week 3 was mainly an intro to flexboxing, in this week we learnt how to flexbox a menu and many different flexbox styles in css. We also learnt about centering images using flexbox and adding videos and audio to our websites, overall the fourth week was an important week in helping to give me the knowledge to start my own website as I had the base knowledge of how to execute my plan.

Week 5 is where we learnt how to use buttons, create footers and also how to use custom fonts.
